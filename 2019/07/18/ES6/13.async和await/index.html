async 和 await

async表示函数里有异步操作，await表示紧跟在后面的表达式需要等待结果

async 和 await 就是将异步代码改为同步代码

async函数的返回值是 Promise 对象, await 后面是一个 Promise 对象

async 是一种函数, await 只能写在 async 中

javascript

async 函数有多种使用形式

javascript

语法

async函数内部return语句返回的值，会成为then方法回调函数的参数

javascript

async函数内部抛出错误，会导致返回的 Promise 对象变为reject状态

javascript

await 后如果不是 Promise 对象。相当于直接返回对应的值

javascript

await 后的 reject 会被catch 接收

javascript

任何一个await语句后面的 Promise 对象变为reject状态，那么整个async函数都会中断执行

javascript

错误处理，将 await 写在try catch 中

javascript

如果多个 await 没有直接关联，可以同时触发，减少耗时

javascript

async 实现原理

async 函数的实现原理，就是将 Generator 函数和自动执行器，包装在一个函数里

async 函数是 Generator 函数的语法糖，将 Generator 函数的星号（*）替换成async，将yield替换成await
