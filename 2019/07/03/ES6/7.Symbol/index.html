概述

Symbol 是一种新增的数据类型，表示独一无二的值

可以接收一个参数，作为对当前Symbol的描述（注释），也可以不传

javascript

相同参数的Symbol值不同

javascript

每个Symbol都是独一无二的

description

用于读取Symbol的描述

javascript

属性名

es5之前对象的属性名只能是字符串，现在可以使用Symbol作为属性名，保证不会出现同名属性

可以为已有对象新增属性，且保证不修改原属性

javascript

访问Symbol属性时只能使用[]

javascript

不能使用点运算符来操作Symbol属性

JS中没有提供隐藏属性，一般以下划线开头表示私有，但还是可以访问到，现在可以使用Symbol来模拟私有属性（symbol的实际应用）

Javascript

属性名的遍历

javascript

Symbol 作为属性名，遍历对象的时候，不会出现在for...in、for...of循环中，也不会被Object.keys()、Object.getOwnPropertyNames()、JSON.stringify()返回。

Symbol.for(),Symbol.keyFor()

Symbol.for()接受一个字符串作为参数，进行全局搜索，如果有以该参数作为名称的 Symbol 值，就返回这个 Symbol 值，否则就新建一个以该字符串为名称的 Symbol 值,并注册到全局

javascript

Symbol(“bar”)不会注册到全局，每次调用都会产生新值

Symbol.keyFor()方法返回一个已登记的 Symbol 类型值的key

Javascript

Symbol.iterator

内置的Symbol 属性，设置遍历器方法
