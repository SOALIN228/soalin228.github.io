Proxy

Proxy 在目标对象之前架设一层“拦截”，外界对该对象的访问，都必须先通过这层拦截，可以对外界的访问进行过滤和改写

Proxy 可以表示为由它来“代理”某些操作，做为“代理器”

javascript

Proxy做拦截，Reflect做赋值

ES6 原生提供 Proxy 构造函数，用来生成 Proxy 实例

new Proxy()表示生成一个Proxy实例，target参数表示所要拦截的目标对象，handler参数也是一个对象，用来定制拦截行为

javascript

如果handler没有设置任何拦截，相当于访问的原对象

javascript

Proxy 实例也可以作为其他对象的原型对象

javascript

proxy对象是obj对象的原型，obj对象本身并没有time属性，所以根据原型链，会在proxy对象上读取该属性，导致被拦截

Proxy 实例的方法

get()

get方法用于拦截某个属性的读取操作，可以接受三个参数，依次为目标对象、属性名和操作的 proxy 实例本身（可选）

可以实现为对象中不存在的属性进行创建，在对象中会添加该属性

javascript

receiver 详解

javascript

如果一个属性不可配置（configurable）且不可写（writable），则 Proxy 不能修改该属性，否则通过 Proxy 对象访问该属性会报错

javascript

set()

set方法用来拦截某个属性的赋值操作，可以接受四个参数，依次为目标对象、属性名、属性值和操作的 proxy 实例本身（可选）

代理 get 和 set 中以下划线开头的自定义私有属性，禁止外部访问

javascript

apply()

apply方法拦截函数的调用、call和apply操作

apply方法可以接受三个参数，分别是目标对象、目标对象的上下文对象（this）和目标对象的参数数组

javascript

has()

has方法用来拦截HasProperty操作,典型的操作就是in运算符

has方法可以接受两个参数，分别是目标对象、需查询的属性名

javascript

construct()

construct方法用于拦截new命令

construct方法可以接受两个参数,分别是目标对象、构造函数的参数对象

Javascript

construct方法返回的必须是一个对象，否则会报错

deleteProperty()

deleteProperty方法用于拦截delete操作，如果方法抛出错误或者返回false，当前属性就无法被delete命令删除

javascript

defineProperty()

defineProperty方法拦截了Object.defineProperty操作

javascript

getOwnPropertyDescriptor()

getOwnPropertyDescriptor方法拦截Object.getOwnPropertyDescriptor()，返回一个属性描述对象或者undefined

javascript

getPrototypeOf()

getPrototypeOf方法主要用来拦截获取对象原型

  • Object.prototype.__proto__
  • Object.prototype.isPrototypeOf()
  • Object.getPrototypeOf()
  • Reflect.getPrototypeOf()
  • instanceof
javascript

isExtensible()

isExtensible方法拦截Object.isExtensible操作

javascript

ownKeys()

ownKeys方法用来拦截对象自身属性的读取操作

  • Object.getOwnPropertyNames()
  • Object.getOwnPropertySymbols()
  • Object.keys()
  • for...in循环
javascript

preventExtensions()

preventExtensions方法拦截Object.preventExtensions()方法

只有目标对象不可扩展时，才可以返回true，否则报错，可以在proxy.preventExtensions方法里面，调用一次Object.preventExtensions

javascript

setPrototypeOf()

setPrototypeOf方法主要用来拦截Object.setPrototypeOf方法

javascript

Proxy.revocable()

Proxy.revocable方法返回一个可取消的 Proxy 实例

javascript

this 问题

目标对象内部的this关键字会指向 Proxy 代理

javascript

Vue3

Vue2 的双向绑定是通过 Object.defineproperty 实现的，他只能对data中已存在的属性进行数据响应式更新，哪怕是null和undefined但是data对象中一定要存在该属性的key,否则无法是实现响应式，要通过Vue提供的this.$set(this.obj,'age',18)完成为新对象属性的添加,实现响应式

Vue3 中将使用Proxy来替代 Object.defineproperty ，因为Proxy可以在不知道对象key的时候进行拦截，所以可以解决上面的问题
