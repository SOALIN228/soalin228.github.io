块级作用域

ES6新增加的作用域，之前只有全局作用域和函数作用域

javascript

匿名立即执行函数表达式也可以被块级作用域取代

Javascript

浏览器为了兼容老代码，块级作用域中的函数会被 var 的形式进行提升，所以推荐使用函数表达式

javascript

ES6 的块级作用域必须有大括号，如果没有大括号，JavaScript 引擎就认为不存在块级作用域，会报错

javascript

let

  1. let作用域在最近的 {} 之间
  2. 如果你在 let a 声明之前使用 a，会报错
  3. 如果你重复 let a，那么报错

var

  1. a = 1，如果当前没有a，声明一个全局 a，如果有 a，为 a 赋值，会造成含义不明

  2. var a，会造成变量提升，会把定义放到最前面，在执行时赋值

  3. var a，可以被重复声明

javascript

const

  1. const 作用域在最近的 {} 之间
  2. 如果你在 const a 声明之前使用 a，会报错
  3. 如果你重复 const a，那么报错
  4. 只用一次赋值机会，且只能在声明时赋值

如果是对象指定的是指针指向的对象不可修改，对象的值可以修改

javascript
