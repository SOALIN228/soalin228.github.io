JavaScript基础知识

JavaScript共有八种数据类型，让我们先来学习一下数据类型的知识吧，只有学好基础知识，在日后的开发工作中，才能做到游刃有余

在HTML中使用JS

  • <script> 标签可以通过 src 属性来设置指向相应文件的URL，这个文件可以是与页面同一个服务器上的文件，也可以是任何域中的文件
  • <script> 中的元素会根据在页面中出现的先后顺序被依次解析，在默认情况下，解析完一个<script>才会解析后面的<script>
  • 默认情况下，解析<script>时，页面中的内容会停止解析，直到<script>解析完毕，所以一般将<script>放在<body>的最下方，这样会减少页面白屏时间提高页面打开速度
  • 使用 defer 属性，可以先将<script>中的内容下载，直到页面内容解析完毕，再去解析脚本，这样可以将<script>写在<head>中，defer 也是按照先后顺序进行解析
  • 使用 async 属性，表示异步解析脚本，但是并不保证解析顺序，所以可以对一些与DOM操作无关的脚本使用 async
  • 使用<noscript>标签来提示用户开启JavaScript

Javascript

严格模式下修复了 js 的一些缺陷，有时候，同样的代码，严格模式执行速度更快

数据类型

JavaScript 的数据类型，有七种基本数据类型和一种复杂数据类型Object

七种基本类型分别为：

  • Undefined
  • Null
  • Boolean
  • Number
  • Bigint(ES10)
  • String
  • Symbol（ES6）

typeof 操作符

typeof 返回一个字符串用于判断数据类型

javascript

nul 的数据类型为 object 是 js 的一个历史bug

也可以使用instanceof 和 Object.prototype.tostring 方法来判断数据的类型

使用typeof 判断除了null之外的基本数据类型，非常好用

实际工作中经常在判断语句中使用typeof ，避免变量未声明而引发报错

Undefined

undefined 表示变量还未进行赋值

未定义的变量不等于未赋值的变量，使用未定义的变量会报错，除非使用typeof

Null

null 表示一个空对象指针，如果在声明变量时都赋值为null 那么就可以判断变量是否已经被引用了

Boolean

boolean 只有true 和false 两个字面值

下面六个值被转为false(falsy)，其他值都视为true

  • undefined
  • null
  • false
  • 0
  • NaN
  • ""或''（空字符串）

Number

浮点数

JavaScript 语言的底层根本没有整数，所有数字都是小数（64位浮点数）。但是由于浮点数不是精确的值，所以涉及小数的比较和运算要特别小心

Javascript

NaN

NaN 是 JavaScript 的特殊值，表示“非数字”，它的数据类型依然属于 number 。任何涉及NaN 的操作，都会返回NaN。NaN 与任何值都不想等，包括他自身

使用isNaN()函数用来判断参数是不是数值,任何不能被转换为数值的参数都会返回true

数值转换

用于把非数值转换为数值，Number() 也可以但是使用复杂，推荐使用下面两个函数

parseInt()

paresInt(0用于将字符串转为整数, 头部空格会被自动去除。空字符串会返回NaN。小数点后部分会被忽略

javascript

parseInt() 存在第二个参数用于指明要解析字符串的格式，默认为十进制

Javascript

推荐无论任何情况下都指明转换格式,避免混淆

parseFloat()

parseFloat() 用于将一个字符串转为浮点数,始终忽略前置0，而且只能以十进制进行解析

String

转义

反斜杠（\）在字符串内有特殊含义，用来表示一些特殊字符，所以又称为转义符

位操作符

  • 二进制或运算符（or）：符号为|，表示若两个二进制位都为0，则结果为0，否则为1。
  • 二进制与运算符（and）：符号为&，表示若两个二进制位都为1，则结果为1，否则为0。
  • 二进制否运算符（not）：符号为~，表示对一个二进制位取反。
  • 异或运算符（xor）：符号为^，表示若两个二进制位不相同，则结果为1，否则为0。
  • 左移运算符：符号为<<。
  • 右移运算符：符号为>>。
  • 头部补零的右移运算符：符号为>>>。

布尔操作符

逻辑非

以下六个值取反后为true，其他值都为false

  • undefined
  • null
  • false
  • 0
  • NaN
  • 空字符串（''）

如果对一个值连续做两次取反运算，变成了与Boolean函数结果相同的布尔值

逻辑与

如果第一个运算子的布尔值为true，则返回第二个运算子的值（不是布尔值）

如果第一个运算子的布尔值为false，则直接返回第一个运算子的值，且不再对第二个运算子求值

取代if结构

javascript

逻辑或

如果第一个运算子的布尔值为true，则返回第一个运算子的值，且不再对第二个运算子求值

如果第一个运算子的布尔值为false，则返回第二个运算子的值

短路规则对这个运算符也适用,

JavaScript 引擎将函数名视同变量名，所以采用function命令声明函数时，整个函数会像变量声明一样，被提升到代码头部,但是函数会先被提升，然后才是变量

函数本身的作用域

函数本身也是一个值，也有自己的作用域。它的作用域与变量一样，就是其声明时所在的作用域，与其运行时所在的作用域无关。

参数传递方式

函数参数如果是原始类型的值（数值、字符串、布尔值），传递方式是值传递。这意味着，在函数体内修改参数值，不会影响到函数外部

javascript

注意，如果函数内部修改的，不是参数对象的某个属性，而是替换掉整个参数，这时不会影响到原始值

arguments 对象

arguments对象包含了函数运行时的所有参数，arguments[0]就是第一个参数，arguments[1]就是第二个参数，以此类推。

javascript

虽然arguments很像数组，但它是一个对象

闭包

如果一个函数，使用了它范围外的变量，那么（这个函数+这个变量）就叫闭包

javascript

闭包的最大用处有两个，一个是可以读取函数内部的变量，另一个就是让这些变量始终保持在内存中，即闭包可以使得它诞生环境一直存在

javascript

闭包的另一个用处，是封装对象的私有属性和私有方法

javascript

注意，外层函数每次运行，都会生成一个新的闭包，而这个闭包又会保留外层函数的内部变量，所以内存消耗很大。因此不能滥用闭包，否则会造成网页的性能问题
