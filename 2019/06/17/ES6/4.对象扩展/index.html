属性的枚举和遍历

枚举

Javascript

enumerable为false时，枚举对象时将会被忽略

  • for...in：只遍历对象自身的和继承的可枚举的属性。
  • Object.keys()：返回对象自身的所有可枚举的属性的键名。
  • JSON.stringify()：只串行化对象自身的可枚举的属性。
  • Object.assign()： 忽略enumerable为false的属性，只拷贝对象自身的可枚举的属性。

遍历

  • for...in：只遍历对象自身的和继承的可枚举的属性，不含（Symbol）
  • Object.keys()：返回对象自身的所有可枚举的属性的键名，不含（Symbol）
  • Object.getOwnPropertyNames():返回对象自身的所有属性（不含 Symbol ，但是包括不可枚举）的键名
  • Object.getOwnPropertySymbols():返回对象自身的所有 Symbol 属性的键名
  • Reflect.ownKeys():返回对象自身的所有键名，不管键名是 Symbol 或字符串，也不管是否可枚举
