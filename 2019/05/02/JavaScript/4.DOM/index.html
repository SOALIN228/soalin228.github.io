Node

属性

nodeType

nodeType属性返回一个整数值，表示节点的类型

  • 元素节点（element）：1 (例如<p>、<div>)
  • 文本节点（text）：3 (文字)

nodeName

nodeName属性返回节点的名称

  • 文档节点（document）：#document

  • 元素节点（element）：大写的标签名

  • 文本节点（text）：#text

textContent

textContent属性返回当前节点和它的所有后代节点的文本内容
